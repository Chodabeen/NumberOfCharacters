# NumberOfCharacters

# H07 -  문자의 개수와 나누어 출력하기
문자열을 입력받아 입력받은 문자의 개수를 출력하고, 각 문자를 한 줄에 하나씩 출력하라. 프로그램은 입력 받는 문자열에 x가 들어오면 종료되도록 한다. 

변수는 다음과 같이 사용하라.

String input; // 입력받는 문자열

int i; // 반복문에 사용되는 변수

